# ERCOT Market Rules Tracker Notes

## 2026-05-18 Initial Build

Project 6 begins as a structured research tracker rather than a live scraper. The starter CSV includes selected ERCOT NPRR items with source URLs, topic labels, priority, and analyst notes.

The important portfolio skill is not claiming to be an ERCOT rules expert. The useful skill is building a repeatable way to read public rule updates, classify possible market impact, and identify questions that should be raised with a trader, scheduler, or operations analyst.

Before any public writeup cites a specific status, the rule should be checked against the current ERCOT source page.
