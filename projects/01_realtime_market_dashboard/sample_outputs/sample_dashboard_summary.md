# Sample Dashboard Summary

Mock-data notice: This summary describes the portfolio sample dataset, not live ERCOT market conditions.

The dashboard allows a user to select a settlement point, choose a date range, and set a price-spike threshold. It then displays real-time price metrics, day-ahead versus real-time comparison, DA/RT spread behavior, load forecast error, and a table of intervals above the selected threshold.

The written summary is generated from the filtered dataset so the dashboard can support both visual review and desk-style reporting.
